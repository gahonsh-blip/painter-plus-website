# Painter Plus – Premium Agency Website

This is a fully static premium website for a painting services business. It uses plain HTML, CSS, and JavaScript and is ready to deploy on GitHub Pages.

## Structure

- index.html – homepage
- about.html – about us page
- services.html – services page
- portfolio.html – portfolio gallery
- contact.html – contact form
- privacy.html – privacy policy
- terms.html – terms of service
- css/style.css – shared styles
- js/script.js – theme toggle, menu, animations
- manifest.json, robots.txt, sitemap.xml, _headers – SEO and PWA support

## Deployment

1. Push the project to GitHub.
2. Open Settings → Pages.
3. Choose the main branch and root folder.
4. Your site will be available at your GitHub Pages URL.

## Customization

Replace contact details, business name, and images as needed. Update the Formspree endpoint in contact.html if you want a live form.
