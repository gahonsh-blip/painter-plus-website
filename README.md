﻿# Painter Plus – Premium Agency Website

This is a fully static, premium business website for a painting services company. It is built with pure HTML, CSS, and JavaScript — no backend, no databases, no Firebase.

## 🚀 Deployment

1. Push all files to a GitHub repository.
2. Go to **Settings → Pages**.
3. Select the branch (e.g., main) and the root folder.
4. The site will be live at https://gahonsh-blip.github.io/painter-plus-website/.

## 📁 Structure

- index.html – Homepage with Hero, Services, Process, Portfolio preview, Testimonials, FAQ, CTA
- bout.html – About Us with Mission, Vision, Values
- services.html – Detailed service cards
- portfolio.html – Project gallery with case studies
- contact.html – Contact form (Formspree) and business info
- privacy.html, 	erms.html – Legal pages
- css/ – All styles (dark/light mode, responsive)
- js/ – Theme toggle, mobile menu, scroll animations, progress bar
- obots.txt, sitemap.xml, _headers – SEO and security

## 🎨 Customisation

- Replace the business name, phone, email, address in all HTML files.
- Replace placeholder images with your own in the images/ folder.
- Update the Formspree endpoint in contact.html with your own form ID.
- Adjust colors in css/style.css if needed.

## 📱 PWA

The site includes a service worker and manifest for offline support and installation on mobile devices.

- index.html – homepage
- about.html – about us page
